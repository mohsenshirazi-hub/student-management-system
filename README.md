# Student Management System
A simple C++ project for managing student information.  
This project demonstrates basic OOP concepts in C++, including:

- Classes & Objects  
- Encapsulation  
- Vectors
- validation
- Input/Output handling  
- Basic data management logic  

## Features
- Add new student  
- List all students  
- Generate unique IDs  
- Search students (optional if implemented)  
- Display student information  

## Technologies Used
- C++
- Standard Template Library (STL)
- g++ or Clang compiler
